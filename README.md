# Asos-Retail-Analysis
  Analysed 15,900+ ASOS products across 338 brands to identify stockout patterns and estimate $472K+ in lost revenue using Python &amp; Pandas


## Problem Statement
ASOS lists products with inline size availability data but no dedicated 
stock column. This project engineers stockout metrics from raw text, 
quantifies the financial impact per brand, and surfaces which brands are 
underperforming on inventory management.

## Key Findings
- $472K+ in estimated lost revenue from out-of-stock sizes
- New Look flagged with 40.9% stockout rate despite being a budget brand
- COLLUSION (53.8%) and Dorina (61.8%) identified as highest risk brands
- Premium brands like Forever and Closet maintain the strongest stock levels

## Tools
Python, Pandas, Matplotlib, Seaborn
